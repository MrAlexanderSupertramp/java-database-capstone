# User Stories

## Doctor
- As a Doctor, I want to view my daily schedule so that I can manage appointments.
- As a Doctor, I want to update my availability times so patients can book slots.

## Patient
- As a Patient, I want to search doctors by speciality so I can find the right provider.
- As a Patient, I want to book appointments online so I don't need to call the clinic.

## Admin
- As an Admin, I want to add and remove doctors so the system reflects staffing changes.
- As an Admin, I want to view reports of appointments so I can manage clinic operations.
